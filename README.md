# ruby-workshop-1

1. Имплементировать абстрактный класс работы с погодой WeatherReporter. В конструктор передаются координаты и другие опциональные kwargs (цельсий/фаренгейт). 
Объект этого класса должен позволять проверить погоду в указанных координатах в переданный момент времени. 
Должен быть хотя бы один реальный источник данных (любое публичное API на своё усмотрение), для каждого источника данных свой класс-потомок. 
Для упрощения или в качестве первого решения можно добавить источник с рандомными данными. 
В тестах не должен вызываться реальный API

```ruby
$ irb
$ require_relative 'weather_reporter'
$ WeatherReporter.new(city: 'Berlin', degrees: :c).call
=> "Current temp in Berlin, Germany is 23C."
```
